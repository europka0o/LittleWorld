Приложение работает только на архитектуре x64 и oc windows, и компиляторе C++ не ниже 14
Проверялось на ос windows 10 и visual studio 2019, на более поздних версиях windows работа не гарантируется 

Сброка спомошью cmake:

1. Клонируй репозиторий
2. В репозитории создай папку с любым названием 
3. Открой командную строку и следуй дальнейшим командам:
    cd Путь до созданной тобой папки в репе
    cmake ..
    cmake -DCMAKE_GENERATOR_PLATFORM=x64 .
    cmake --build . --config release
4. После этих команд в папке должен создасться и забилдиться проект
5. Открой папку, где был построен exe'шник, и закинь туда папки Sounds и Img, а так же 
следующие dll из папки SFML-2.5.1\bin :
    openal32.dll 
    sfml-audio-2.dll
    sfml-graphics-2.dll
    sfml-system-2.dll
    sfml-window-2.dll

Вуаля! Приложение собрано и готово к работе
